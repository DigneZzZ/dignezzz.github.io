# SingBox Configs

Конфигурации маршрутизации и VPN-профили для SingBox.

| Файл | Описание |
|------|----------|
| **ru_singbox.json** | Конфигурация SingBox с правилами маршрутизации для RU-сегмента |
| **ru_singbox_2.json** | Альтернативная конфигурация SingBox |
| **ru_direct.conf** | Список доменов для прямого подключения (без VPN) |
| **ru_direct2.conf** | Альтернативный список прямых доменов |
| **ru_direct3.conf** | Расширенный список прямых доменов |
