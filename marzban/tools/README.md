# Marzban Tools

Утилиты для управления ядром и миграции базы данных Marzban.

| Скрипт | Описание | Установка |
|--------|----------|-----------|
| **core_change.sh** | Смена ядра прокси (Xray core) с преобразованием форматов конфигурации | `bash <(wget -qO- https://dignezzz.github.io/marzban/tools/core_change.sh)` |
| **migrate_to_mysql.sh** | Миграция базы данных Marzban на MySQL с поддержкой нескольких БД | `bash <(wget -qO- https://dignezzz.github.io/marzban/tools/migrate_to_mysql.sh)` |
