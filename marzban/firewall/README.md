# Marzban Firewall

Настройка правил файрвола UFW специально для серверов Marzban.

| Скрипт | Описание | Установка |
|--------|----------|-----------|
| **ufw_main.sh** | Конфигурация UFW для основного сервера Marzban | `bash <(wget -qO- https://dignezzz.github.io/marzban/firewall/ufw_main.sh)` |
| **ufw_node.sh** | Конфигурация UFW для узла (node) Marzban | `bash <(wget -qO- https://dignezzz.github.io/marzban/firewall/ufw_node.sh)` |
